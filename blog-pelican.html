Pelican 重新搭建我的博客

很长时间没有更新博客了，一方面没有时间来写博客，另一方面，对原来的博客并不是很满意，所以打算重新搭建一个静态博客。由于最近一直使用的是 Python，所以打算使用 Pelican 来搭建新的博客。

快速搭建静态网站

可以查看 Pelican 4.2.0 中文文档 或者 Pelican 4.5.2官方文档 文档寻求帮助。

首先，使用 pip 工具安装 Pelican，如果没有安装 markdown 的还需要安装 markdown。

pip3 install pelican
pip3 install markdown

安装完成后，新建一个项目，然后在这个项目的目录中使用 pelican-quickstart 命令初始化项目框架。

$ pelican-quickstart
Welcome to pelican-quickstart v4.2.5.

This script will help you create a new Pelican-based website.

needed by Pelican.

> Do you want to specify a URL prefix? e.g., https://example.com   (Y/n) n
> Do you want to enable article pagination? (Y/n) y

中文只能通过手动修改根目录下的 pelicanconf.py 文件进行设置，将 DEFAULT_LANG 设置为 zh_cn 即可。

这样，一个静态网站框架就搭建好了，只需要将写好的博客放在 content 文件夹下，使用 pelican content 命令即可以生成静态网页，再使用 pelican --listen 启动 Web 服务登录到 http://localhost:8000 查看博客的效果。

Pelican 博客主题

登录 Pelican 主题网站 可以查看 Pelican 博客的各种主题，选择自己喜欢的 Pelican-Blue 主题，将其克隆到本地。

$ git clone https://github.com/Parbhat/pelican-blue.git

进行相关的配置：

THEME = 'themes/pelican-blue'
SIDEBAR_DIGEST = '侧边栏摘要'
FAVICON = 'url-to-favicon'
DISPLAY_PAGES_ON_MENU = True

然后重新生成即可再网页中看到不一样的效果。

社交

模板中的社交提供了 facebook，github，linkedin，twitter 和 mail，而这其中常用的只有 Github，所以添加个人的 Github 主页。而常用的比如 CSDN 博客，博客园，微博等社交账号并没有，所以需要自己添加社交帐号的图标到主题的 static/images/icons/ 中，然后添加自己的社交账号到 Social 中即可。以微博为例，

# Social widget
SOCIAL = (('github', 'https://github.com/name'),
          ('weibo', 'https://weibo.com/name'),)

更新后左边就有了社交网站添加的图标。

数学公式渲染

我所要做的是一个主要分享数学的博客，所以不可避免会遇到各种数学公式，这就需要使用数学公式的渲染。想要实现数学公式渲染有两种方式，一种是安装 render-math 插件。查看官方文档，根据 README 的说明需要先安装 typogrify 库：

pip3 install typogrify

安装完成后，再项目的根目录下新建文件夹用来保存插件，并将 render-math 插件克隆到这个文件夹。

git clone https://github.com/barrysteyn/pelican_plugin-render_math.git

然后修改配置文件 pelicanconf.py：

PLUGIN_PATHS = ['plugins']
PLUGINS = ['pelican_plugin-render_math']

重新生成页面并在浏览器中观看渲染的效果。第一次渲染的速度有点慢，渲染的结果也不是太好看，当然可以改变配置更加完善。

第二种方式是直接修改主题的 base.html 文件，将下面支持数学公式的代码插入到 <head> 标签中，然后即可支持数学公式的渲染。

<script type="text/javascript" src="https://cdnjs.cloudflare.com/ajax/libs/mathjax/2.7.1/MathJax.js"></script>
<script>
    MathJax.Hub.Config({
        config: ["MMLorHTML.js"],
        extensions: ["tex2jax.js"],
        jax: ["input/TeX", "output/HTML-CSS", "output/NativeMML"],
        tex2jax: {
            inlineMath: [ ['$','$'], ["\\(","\\)"] ],
            displayMath: [ ['$$','$$'], ["\\[","\\]"] ],
            processEscapes: true
        },
        TeX: {
            extensions: ["AMSmath.js", "AMSsymbols.js"],
            TagSide: "right",
            TagIndent: ".8em",
            MultLineWidth: "85%",
            equationNumbers: {
                autoNumber: "AMS",
            },
            unicode: {
                fonts: "STIXGeneral,'Arial Unicode MS'"
            }
        },
        displayAlign: "center",
        showProcessingMessages: false,
        messageStyle: 'none'
    });
</script>

由于上述代码使用的是 CDN 服务器上的 MathJax，所以访问可能不稳定，也可以自己在本地部署一个 MathJax，具体可以参考使用 Markdown + MathJax 在博客里插入数学公式。

添加评论

Pelican 静态博客是支持 Disqus 评论的，申请 Disqus 账号然后在 pelicanconf.py 文件中添加如下配置语句即可：

DISQUS_SITENAME = "{Shortname}"

但是，Disqus 必须科学上网才能使用。在最开始搭建 Hexo 静态博客的时候，我使用了 GitHub Issues 的评论系统 Gitment，用了这么长时间感觉还可以，所以我依然想继续使用 Gitment 作为我的评论系统。

Pelican 的主题模板中并没有配置 Gitment 评论，所以需要在 article.html 中添加以下内容：

{% if GITMENT_SITENAME %}
    <div class="comments" id="comments">
    <!--汉化-->
    <link rel="stylesheet" href="https://billts.site/extra_css/gitment.css">
    <script src="https://billts.site/js/gitment.js"></script>
    <!--原型-->
    <!--
    <link rel="stylesheet" href="https://imsun.github.io/gitment/style/default.css">
    <script src="https://imsun.github.io/gitment/dist/gitment.browser.js" type="text/javascript"></script>
    -->
    <div id="gitmentContainer" style="margin-bottom: -19px;"></div>
    <style>
        .gitment-container a {
        border: none;
        }
        .comments {
        margin: 60px 0 0;padding: 0 60px;
        }
    </style>
    <script type="text/javascript">
        var gitment = new Gitment({
            id: '<%= article.title %>',
            title: '<%= article.title %>',
            owner: '{{ GITMENT_SITENAME }}',
            repo: '{{ GITMENT_SITENAME }}.github.io',
            oauth: {
            client_id: '{{ GITMENT_ID }}',
            client_secret: '{{ GITMENT_SECRET }}',
            },
        })
        gitment.render('gitmentContainer')
    </script>
    </div>
{% endif %}

这部分内容是我修改了 Hexo 博客的代码，否则直接复制是无法使用的。然后在 pelicanconf.py 文件中添加以下配置：

GITMENT_SITENAME = 'yoursitename'
GITMENT_ID = '************'
GITMENT_SECRET = '************'

重新生成页面，就可以看到文章最下面的评论了。

为源代码加密

由于有一些配置涉及到了账号密码等信息，而这些信息是可以通过查看源代码看到的。但对源代码进行加密是比较复杂的，所以考虑禁止查看源代码。查看源代码有几种方式，分别是右键点击查看源代码，快捷键 Ctrl + U 或者 Ctrl + Shift + I 查看源代码以及 F12。

禁止右键和禁止复制实现比较简单，在 <body> 标签中加入下列内容：

<body oncontextmunu="return false" onselectstart="return false">

参考网站禁止右键、F12“防复制、防查看源代码、防图片下载”方法，禁止 F12 可以通过以下方式实现：

<script type="text/javascript">
    document.onkeydown = function () {
        if (window.event && window.event.keyCode == 123) {
                event.keyCode = 0;
                event.returnValue = false;
                return false;
            }
        };
</script>

禁止 Ctrl + U 和 Ctrl + Shift + I 这两种方式，参考JS禁止查看网页源代码的简单实现方法中的内容，然后修改了上述禁止 F12 操作的代码成功实现。现在这个博客就是禁止复制，禁止右键，禁止查看源代码的。当然，这些是防不住编程高手的。
