搭建更美观的博客

选择主题

进入Hexo主题专栏，从中选择自己喜欢的博客主题然后进行克隆。我选择了有哪些好看的 Hexo 主题这篇中的 Huno 主题。

复制相关文件在 GitHub 上的地址，然后进行克隆：

git clone git://github.com/someus/huno.git themes/huno

下载完成后配置文件 _config.yml，找到关键字 theme，修改参数为：theme: huno，然后部署主题：

hexo g
hexo s

感觉不错，清除缓存，将其部署到 GitHub 上：

hexo clean
hexo g -d

此时访问博客就可以看到更换后的主题。

另外，还可以修改 \themes\hexo 文件夹下的配置文件 _config.yml，进行个性化的设计。

归档页面

归档页面会显示分类、标签云以及基于日期的归档。

在主题的配置文件中：

# Header
menu:
  首页: /#blog
  关于: /about
  归档：/archive

创建新的 page：

hexo new page archive
cd source/archive
vim index.md

内容修改为：

title: 归档
layout: page-archive
---

Social Icon

默认提供 GitHub 图标，用户名在 Hexo 的配置文件中配置，例如：

# Social
social:
  github: your Github name

可以根据需要在 layout/_partials/social.ejs 中添加更多的图标。

访客统计

使用不蒜子进行网站计数，在当前主题文件夹下的 layout\_partial\footer.ejs 文件中加入以下脚本：

<script async src="//busuanzi.ibruce.info/busuanzi/2.3/busuanzi.pure.mini.js"></script>

然后加入以下内容：

本站总访问量<span id="busuanzi_value_site_pv"></span>次
本站访客数<span id="busuanzi_value_site_uv"></span>人次
本文总访问量<span id="busuanzi_value_page_pv"></span>次

这段内容我放在了 <footer> 和 </footer> 之间，这样的话显示的字比较小。

更多相关内容请访问搞定你的网站计数。

图标

找到合适的图标，放在 themes/theme/source/images/ 文件夹中，并修改主题配置文件即可。

评论

Huno 主题自身支持 Disqus 和多说，但是多说已经停止服务，Disqus 在国内无法使用。所以我找到了一篇 Hexo 添加 Gitment 评论 文章，并尝试添加。

首先注册 Gitment OAuth Application，填入评论页面的域名然后得到一个 client ID 和 client secret，用于之后的用户登录。

安装 Gitment 插件

使用 npm 安装确定版本的 Gitment

npm install --save gitment

引入 Gitment

将下面代码添加到文章页面

<!--gitment 评论-->
  <div class="comments" id="comments">

    <!--汉化-->
    <link rel="stylesheet" href="https://billts.site/extra_css/gitment.css">
    <script src="https://billts.site/js/gitment.js"></script>
    <!--原型-->
    <!--
    <link rel="stylesheet" href="https://imsun.github.io/gitment/style/default.css">
    <script src="https://imsun.github.io/gitment/dist/gitment.browser.js" type="text/javascript"></script>
    -->

      <div id="gitmentContainer" style="margin-bottom: -19px;"></div>

      <style>
        .gitment-container a {
          border: none;
        }
        .comments {
          margin: 60px 0 0;padding: 0 60px;
        }
      </style>

      <script type="text/javascript">
        var gitment = new Gitment({
              id: '<%= page.title %>',
              title: '<%= page.title %>',
              owner: 'your Github name',
              repo: 'yourgithubname.github.io',
              oauth: {
                client_id: '************',
                client_secret: '************',
              },
        })
        gitment.render('gitmentContainer')
      </script>

  </div>
<!--gitment 评论 end-->

其中：

  • id: ‘<%= page.title %>’,// 页面标题作为 issue 的 label，长度不超过50，否则会出现 Error：validation failed 错误提示
  • owner 你的GitHub用户名即可
  • repo 保存评论的GitHub仓库名
  • client_id 你注册的Gitment Client ID [可在developers找到你注册的Gitment查看]
  • client_secret 你注册的Gitment Client secret [可在developers找到你注册的Gitment查看]
  • 汉化来自评论框汉化问题

主题配置

在主题的配置文件中添加以下内容：

# Gitment
# Introduction: https://imsun.net/posts/gitment-introduction/
gitment:
  enable: true
  githubID: your Github name
  repo: yourgithubname.github.io
  ClientID: ************
  ClientSecret: *************
  lazy: false

其中 lazy 表示评论懒加载，即不直接显示评论，false 则直接显示评论。

初始化评论

发布页面后，访问网页并使用自己的 Github 账户登录，点击初始化按钮。

注意：保证登录的账号是配置中的 repo 和 owner。只有自己先初始化后才会在对应的 Github 仓库的 issue 中创建一条对应的 issue，之后其他用户即可在该页面发表评论。

现在就可以看到评论你的评论了。
